The Project

This is an desktop client for the microblogging web service Twitter.

Features already implemented

With this client you can enjoy the full range of functionalities Twitter offers without need to visit the website. You can manage multiple accounts simultanously [not yet implemented].

List

  • Home timeline
  • Retweet
  • User Details
  • [Un]Follow Users

Intention

The intention of the project is to implement a sample application for Twitter4J.

Technical Data

  • implemented with: Java SE 7
  • Graphical User Interface: Java Swing Library
  • Twitter4J v3.0.3

System Requirements

  • JRE 7 installed (download)
  • existing internet connection
