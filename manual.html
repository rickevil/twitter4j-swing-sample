How to use the client

Installation

  1. Download the application
  2. Download the file login.xml and save it right beside the application
  3. Go to dev.twitter.com
    • log in there
    • create an application
    • set the access level to "Read, Write & Direct Messages"
    • create the "Access Token"
    • copy & paste "consumer key", "consumer secret", "access token", and "access token secret" into the login file

Then you can start to tweet!
