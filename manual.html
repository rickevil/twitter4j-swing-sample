How to use the client

Installation

  1. Download the application
  2. Download the file login.xml and save it right beside the application
  3. Go to dev.twitter.com
    • log in there
    • create an application
    • set the access level to "Read, Write & Direct Messages"
    • create the "Access Token"
    • copy & paste "consumer key", "consumer secret", "access token", and "access token secret" into the login file
    You can add more than one of your accounts this way by copying the !

Then you can start to tweet!

Usage

How to use the client

# Switching between accounts

To switch between access data you have entered into login.xml click on 'File' → 'Switch user' → '[user name]'

# Load Home Timeline

In Home Timeline Panel click on 'load Home Timeline' to load the latest tweets

# Displaying some information about the user

In any timeline click on the profile image of the user you want to get some information about

# Write a tweet

To write a tweet, click on the write button of the toolbar on the left, write your tweet, and press the 'tweet' button

# Searching tweets

If you want to search for special tweets, click on the search button of the toolbar on the left, type in your query, and press the 'search' button
